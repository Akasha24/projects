## How to run locally

1. Open Visual Studio Code
2. Open any folder where you want the project
3. Open the terminal in VS Code
4. Run the following command:

   git clone https://github.com/Akasha24/projects.git

5. After cloning is complete, open the project folder
6. Right-click `index.html` and open it using Live Server

The website will open automatically in your browser.
